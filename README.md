# Mobile Resale Price Predictor

A simple Machine Learning model using Logistic Regression to predict if a used mobile can be sold at a particular price.

**Accuracy  = 95.33<br/> 
Epoch       = 10000<br/>
Alpha       = 0.002<br/>**
